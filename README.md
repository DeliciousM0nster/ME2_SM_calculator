# ME2_SM_calculator
Who will survive the ME2 Suicide Mission? Calculate it!

Notes:
  1) Can run ME2_suicideMission_outcomePredictor.py from command line. Edit choices with IDE or whatever.
  2) All options that effect the outcome are under "Flags and Options" section at the top of the code.
  3) Notes on what are allowed for each variable are in the code.
  4) ME2_suicideMission_outcomePredictor_withGUI.py is the same program but with a GUI and more restraints.
  5) For example, you can't take both Morinth and Samara with you, you can't select the a person for both teh vents and your squad, etc.
  6) I tried my best to make sure it reflected the limitations the game would put on you (like who is available as a biotic or tech specialist, etc)
